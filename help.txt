Функции для работы с атрибутами
-------------------------------
В Python существуют встроенные функции, которые используются для работы с атрибутами. Самые распространённые из них: 

setattr - создание атрибута или изменение его значения
getattr - получение значения атрибута
hasattr - проверка наличия атрибута
delattr - удаление атрибута
